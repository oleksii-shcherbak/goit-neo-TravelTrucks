# TravelTrucks

Web application for camper rental services in Ukraine.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/oleksii-shcherbak/goit-neo-TravelTrucks
cd goit-neo-TravelTrucks
```

2. Install dependencies:
```bash
npm install
```

## Running the Project

### Development Mode
```bash
npm run dev
```
The application will be available at `http://localhost:5173`

### Production Build
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Tech Stack

- React 18.3.1
- Vite 5.4.1
- Redux Toolkit 2.2.8
- React Router DOM 6.27.0
- Tailwind CSS 3.4.14
